# Techcombank — Product Engineering Playbook

Short, production-ready repo of artifacts a Senior Product Owner at Techcombank would use.

- **Purpose:** central place for product docs, roadmaps, specs, visuals, and release/monitoring artifacts for Techcombank retail & business products.
- **Audience:** senior PO, PMs, EMs, stakeholders at Techcombank.
- **Repo outputs:** product spec templates, OKRs, roadmap, release plan, monitoring checklist, stakeholder map, persona & journey, backlog sample, mermaid visuals.

Quick start — set your git author then commit & push:

```bash
git config user.name "Your Name"
git config user.email "you@example.com"
git init
git add .
git commit -m "chore: add product engineering playbook"
# create remote on GitHub (replace with your repo)
# Option A: create repo manually on GitHub then:
git remote add origin git@github.com:lamtquyen03/techcombank-product-engineering.git
git branch -M main
git push -u origin main
```
